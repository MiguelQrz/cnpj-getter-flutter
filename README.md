# 📱 Flutter CNPJ Getter

Um aplicativo desenvolvido em **Flutter** que permite buscar informações de empresas brasileiras a partir do **CNPJ**, utilizando a API pública do **OpenCNPJ**.

---

## 🚀 Funcionalidades

* 🔎 Consulta de empresas por CNPJ
* 📄 Exibição de dados cadastrais, como:

  * Razão social
  * Nome fantasia
  * Situação cadastral
  * Data de abertura
  * Endereço completo
  * Atividade principal
* ⚡ Interface simples e responsiva
* 🌐 Integração com API REST (OpenCNPJ)

---

## 🛠️ Tecnologias utilizadas

* Flutter
* Dart
* HTTP (consumo de API)
* OpenCNPJ API

---


## 📱 Exemplo de uso

1. Digite um CNPJ válido no campo de busca
2. Clique em **CONSULTAR**
3. Visualize os dados retornados pela API

---


## ⚠️ Observações

* Certifique-se de usar um CNPJ válido
* A API pode ter limites de requisição
* Dados retornados dependem da disponibilidade do OpenCNPJ
